## What Is This Repository?

Welcome to my guide repository for **CPCS203: Programming-II**!

Instead of posting raw code, this repository contains **step-by-step breakdowns**, **architectural blueprints**, and **logical approaches** to solving our Java Object-Oriented Programming (OOP) assignments. The goal is to help you understand the **how** and the **why** behind the code so you can build the solutions yourself.

> ⚠️ **Note to Students:** You won't find `.java` files to copy and paste here. Use these guides as a **map** to write your own code, understand the core concepts, and ace your evaluations while maintaining academic integrity.

## What's Inside Each Guide?

For every assignment, the dedicated `.md` file includes:

1. **The Blueprint** — A breakdown of the required classes, objects, and UML structure.
2. **Data Structures** — Explanations of how to set up arrays or collections.
3. **Core Logic** — Step-by-step algorithms for the tricky parts.
4. **Formatting Tips** — Hints on how to perfectly match required output files using `PrintWriter` and `printf`.
5. **Final Checklist** — A quick self-review list before submission.

---

## Core OOP Concepts Covered

- **Encapsulation** — Private fields with public getters/setters
- **Class Design** — Constructors, relationships between objects
- **File I/O** — Reading from `input.txt`, writing to output files with `PrintWriter`
- **Data Structures** — 1D arrays, 2D arrays, and Ragged Arrays
- **Control Flow** — `while` loops, `switch` statements for command processing
- **String Formatting** — `printf` with format specifiers for aligned output
